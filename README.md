# 충전소 찾기 (EV Charger Finder)

내 주변과 **가는 길**의 전기차 충전소를 네이버 지도에서 **실시간 빈자리**와 함께 보여주는 모바일 웹앱. 테슬라 슈퍼차저는 따로 강조한다.

🔗 **사용하기: https://ev-charger-finder-five.vercel.app** (모바일 Safari/Chrome 권장)

---

## 사용 방법

- **앱 열기** → 위치 권한을 허용하면 내 위치 주변 충전소가 바로 뜬다. (오른쪽 아래 📍 버튼으로 다시 내 위치로)
- **마커 색**
  - 🟢 초록 = 지금 빈자리 있음 (숫자 = 사용 가능한 대수)
  - ⚪ 회색 = 만차 / 점검 중
  - 🔴 빨강 ⚡ = 테슬라 슈퍼차저
  - 🔵 파란 점 = 내 위치
- **검색**: 위 검색창에 장소·지명·주소 입력 (예: `강남역`) → 그 위치로 이동하고 핀이 고정된다.
- **필터 칩**: 전체 / 급속 / 완속 / 테슬라 / 빈자리(있는 곳만).
- **가는 길 충전소**: 목적지를 검색한 뒤 **"가는 길 충전소"** 버튼을 누르면, 내 위치→목적지 경로선과 함께 **경로 ±1km 안의 고속도로 휴게소 급속만** 먼저 보여준다. (여행용) 배너의 **"휴게소 외 급속도 보기"** 버튼으로 경로상 일반 급속까지 펼칠 수 있다. (휴게소가 없는 짧은 경로는 자동으로 전체 급속을 표시한다.)
- **마커를 누르면**: 이름·실시간 빈자리·커넥터·운영기관·거리·마지막 갱신 시각이 팝업으로 뜨고, **"네이버지도로 길찾기"** 를 누르면 내 위치→그 충전소 경로가 네이버 지도에 바로 표시된다.
- **하단 리스트**: 주변 충전소를 이름·빈자리·거리로 한눈에. 탭하면 해당 충전소로 이동.
- **2D ↔ 상세(위성) 토글** 버튼, 휠/트랙패드 줌(민감도 조절됨).

> 실시간 "빈자리 수"는 보여주지만, 개별 차량의 "충전 잔여 시간"은 공개 데이터에 없어 제공하지 않는다.

## 데이터 출처

- **충전소 위치·실시간 상태**: 공공데이터포털 — 한국환경공단 전기차 충전소 정보 (`getChargerInfo`)
- **테슬라 슈퍼차저**: supercharge.info (한국, 운영중 사이트)
- **지도 / 주소·장소 검색 / 역지오코딩**: 네이버 지도 (NCP) + 네이버 지역검색 Open API
- **경로(가는 길)**: OSRM (오픈소스 라우팅, 공개 서버)

## 구조 (개발자용 요약)

```
public/        정적 프론트엔드 (네이버 지도 + UI)
api/           Vercel 서버리스 함수
  chargers.js  KV에서 시군구별 충전소 읽기 (사전빌드 캐시, 콜드 호출 없음)
  search.js    네이버 지역검색 프록시
  route.js     OSRM 경로 프록시
lib/           순수 로직 (거리·분류·파싱·집계·저장소 추상화) — node --test
scripts/
  build-all.mjs       시도 단위로 전국 충전소를 받아 시군구별로 KV에 사전빌드
  warm.mjs            특정 시군구 갱신(정기 워머용)
  fetch-tesla.mjs     테슬라 슈퍼차저 데이터 갱신
```

### 왜 KV 사전빌드인가
공공 API가 호출당 30~60초로 느려서, 앱이 요청 때마다 직접 부르면 느리고 Vercel 타임아웃이 난다. 그래서 **백그라운드에서 미리 받아 Upstash KV에 저장**하고, 앱은 KV에서 즉시 읽는다.

### 로컬 실행
```bash
npm test                                  # 로직 테스트 (node --test)
node --env-file=.env.local scripts/dev-server.mjs   # 로컬 서버 (localhost:3000)
```

### 환경변수 (서버 전용 — .env.local / Vercel, 절대 커밋 금지)
- `DATA_GO_KR_KEY` — 공공데이터포털 인증키
- `NAVER_SEARCH_CLIENT_ID`, `NAVER_SEARCH_CLIENT_SECRET` — 네이버 지역검색
- `KV_REST_API_URL`, `KV_REST_API_TOKEN` — Upstash KV
- (네이버 지도 Client ID는 도메인 제한으로 보호되는 공개값이라 프론트엔드에 포함)

### 데이터 갱신 / 배포
```bash
node --env-file=.env.local scripts/build-all.mjs 11,28,41   # 시도별 사전빌드
vercel deploy --prod                                        # 배포
```

## 알려진 한계 / 다음 단계

- **빈자리 신선도**: 현재 KV는 사전빌드 시점 기준이다. 정기 워머(맥미니/GitHub Actions cron)를 걸면 분 단위로 최신 유지된다. (예정)
- **경로**: OSRM 공개 서버를 쓰므로 가끔 느릴 수 있다.
- **주차장 레이어 (v2 예정)**: 공공 주차장 데이터와 충전기를 교차해 "이 주차장에 충전기 있음"을 표시.
